H1为什么使用 Effie ？

Effie 是思想家专用的写作软件。

即使是优秀的思想，也只有变成文字才能为人所知，这就是思想家的宿命。

Effie 致力于把思想变成文字，最终成为价值。

 

Effie 的软件设计，是基于对思想家的如下理解：

H2思想家是心无旁骛的

  • Effie 是激进的极简主义现场
    •  没有主色调
    •  界面几乎没有按钮
    •  设计即退让
  • 帮助思想家摈弃一切干扰，专注在创作本身
  • “不觉已春深，物我两相忘”

H2思想家是注重逻辑的

  • Effie 是冷静和理性的代表作
    •  以更有逻辑的方式展示列表
    •  让凌乱的思绪轻松归位
    •  一键生成思维导图
  • 帮助思想家把复杂的思想，展现成清晰的结构
  • “桐花诗八韵，思绪一何深”

H2思想家是敏捷高效的

  • Effie 是提高效率的利器
    •  无需文件管理，输入文章标题等操作
    •  云端同步，没有后顾之忧
    •  双手不离键盘，即可调整样式
  • 帮助思想家尽情输出，释放创作激情
  • “无穷如天地，不竭如江海”

愿 Effie 成为您手中的创作利器。

有意清秋入衡霍，为君无尽写江天。

H1Effie 使用入门

H2开始创作

在左侧面板管理您的文稿。用鼠标左键拖动，用鼠标右键菜单下达指令。

您可以随意移动文稿和排序。

点击左上的+ 新建文稿创建新的文稿。

点击左上角的 << 可以收起左边栏，进入完全无干扰的写作环境。

H2使用列表和思维导图

列表可以帮您更清晰的整理思路和逻辑。在文稿中输入 - 或 + 然后空格，即可进入列表模式。

  • 在列表模式中，按 Tab 键和 Shift+Tab 键 进行缩进。
  • 用鼠标点按圆点可以展开和收起子项列表。
  • 点击列表右侧的 ⑆ 按钮进入思维导图模式。
  • 思维导图模式同样支持编辑、拖动、展开、收起等操作。

H3试试看

  • 试试点击本行左侧圆点
    •  项目1
    •  项目2
  • 试试点击列表右侧的 ⑆ 按钮
    •  试试在思维导图模式编辑和操作
  • 试试在本行末尾敲回车输入文字，然后按 Tab 和 Shift+Tab

H2文稿样式

H3输入标题

在行首输入 # 然后空格，就可以输入标题了。

在行首输入 ## 空格是二级标题，以此类推，最多六级。

H3给文字添加样式

在文字前后加一个星号是斜体，加两个星号是粗体，加两个冒号是标注，加两个波浪线是删除，加两个加号是。

“在行首输入 > 然后空格，添加引用。

输入三个减号并回车，即可插入分隔线。

H3插入代码

在文字前后加一个重音符号（数字1左边的按键）表示代码，如：make_girls_happy.h

 

在行首输入两个重音符号并空格表示代码块，如： 
</>#include <stdio.h>
int main() {
   printf("Hello, Thinker!");
   return 0;
}

H3转义字符

输入 * _ 等符号时，如不想触发样式，请在前面加 符号。

H2把文稿导出成常见格式的文件

在左侧预览卡片上点击右键，选择导出，即可导出常见文件格式。

 

如需一次导出多个文件，可在左侧预览卡片选中多个文件，点击右键，选择导出。

H1Effie 快捷键列表

H2基本操作

  • ⌘1 展开/收缩左边栏
  • ⌘N 新建文稿
  • ⇧⌘N 新建组
  • ⇧⌘F 全局搜索
  • ⌘, 打开偏好设置
  • ⌥⌘↑ 打开上一篇文稿
  • ⌥⌘↓ 打开上一篇文稿
  • ⌃⌘F 进入/退出全屏幕
  • ⌘H 隐藏Effie
  • ⌘W 关闭窗口
  • ⌘Q 退出Effie

H2大纲

  • ⌘` 打开思维导图
  • ⇥ 向前缩进
  • ⇧⇥ 向后缩进

H2思维导图

  • ⌘+ 放大思维导图
  • ⌘- 缩小思维导图
  • ⌘0 还原思维导图默认大小
  • ↩︎ 插入主题
  • ⇥ 插入子主题

H2编辑区快捷键

  • ⌘A 全选
  • ⌘C 拷贝
  • ⌘X 剪切
  • ⌘V 粘贴
  • ⌘Z 撤销
  • ⇧⌘Z 重做
  • ⌘B 加粗
  • ⌘I 斜体
  • ⌘L 清除标记
  • ⌘F 查找
  • ⌘G 查找下一个
  • ⇧⌘G 查找上一个
  • ⌥⌘F 查找和替换
  • ⇧⌘↑ 选中插入点与文稿开头之间的文本
  • ⇧⌘↓ 选中插入点与文稿末尾之间的文本
  • ⌃F 光标向右移动一个字符
  • ⌃B 光标向左移动一个字符
  • ⌃P 光标向上移动一行
  • ⌃N 光标向下移动一行
  • ⌃A / ⌘← 光标移动到当前行行首
  • ⌃E / ⌘→ 光标移动到当前行行尾
  • ⌘↑ 光标移动到文章开头
  • ⌘↓ 光标移动到文章末尾
